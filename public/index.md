# Email Agent Skill

emailagentskill.com

Email Agent Skill for agents that work on email systems.

A compact skill for agentic email planning, tool handoffs, safe execution, and review loops, built for Codex, Hermes, OpenClaw, Claude Code, Cowork, Goose, Cursor, Windsurf, and other skill-aware agents. It gives agents intake questions, review gates, QA checks, and handoff formats for production email work.

## Install

```bash
npx skills add emailagentskill
```

Raw skill folder: `skills/emailagentskill`

## Use Cases

- **Agent task routing**: Give agents a bounded way to inspect inputs, propose changes, and surface approval points before anything reaches subscribers.
- **Approval gates**: Give agents a bounded way to inspect inputs, propose changes, and surface approval points before anything reaches subscribers.
