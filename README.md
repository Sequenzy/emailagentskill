# Email Agent Skill

Static landing page and agent skill for emailagentskill.com. Works with Codex, Hermes, OpenClaw, Claude Code, Cowork, and other skill-aware agents.

Install shortcut:

```bash
npx skills add emailagentskill
npx skills add Sequenzy/emailagentskill --skill emailagentskill
```

Raw skill folder: `skills/emailagentskill`
